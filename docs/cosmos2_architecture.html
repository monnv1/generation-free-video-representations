关键洞察

为什么 t=0 还能工作？
DiT 在 timestep=0 时退化为一个 带 text-conditioning 的时空 self-attention 编码器。 Self-attention 仍然建模 latent patch 之间的时空依赖（物体运动、空间关系），3D RoPE 位置编码完全保留。 Cross-attention 仍然让 latent (Q) 查询 text embeddings (K,V)，语言信息照样注入。 预训练阶段学到的物理直觉、运动先验被完整保留在权重里，一个 clean pass 就能提取。
starVLA 本质是什么？
把一个预训练的 video diffusion model 当作冻结的时空视觉 backbone — 和把 CLIP/DINO 作为 vision encoder 的思路一样，不同的是 Cosmos 的 DiT 内在编码了物理动态、 3D 时空关系、以及语言 grounding。不需要扩散采样，单次 forward 就能榨出 rich spatiotemporal features。
WMActionResampler 的角色
DiT 输出 ~4700 个 token × 2048 维，直接喂给 action head 太重。 Resampler (7.92M, Perceiver/Q-Former 风格) 用 64 个 learnable queries 通过 cross-attention 将长序列压缩为 64×512 的 action condition tokens，再交给 FlowmatchingActionHead 做 flow-matching 动作预测。
